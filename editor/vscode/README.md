## What's Included

Visual Studio Code productivity tips:
- **Multi-cursor editing** shortcuts
- **Column selection** techniques
- **Regex find & replace** across solution
- **Refactoring** workflows (F2 rename)

## VSCode Shortcuts

### Multi-Cursor Editing
- `Alt + Click`: Add cursor at click position
- `Alt + Shift + ↑ / ↓`: Add cursor above/below (column mode)
- `Alt + ↑ / ↓`: Move line up/down

### Find & Replace
- `Ctrl + Shift + H`: Find and replace (with regex support)
- Works across entire solution
- Enable "Match whole word" and "Match case" for precision

### Refactoring
- `F2`: Rename symbol (works across solution)
- Shows preview of all changes
- Safe refactoring with undo support

## Tips

1. **Use multi-cursor for repetitive edits**: Select similar patterns, edit once
2. **Regex is powerful**: Learn basic patterns for find/replace
3. **Rename safely**: Always use F2 for symbol renaming, not find/replace
4. **Preview changes**: Review refactoring previews before applying
