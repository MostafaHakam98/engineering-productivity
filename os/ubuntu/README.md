# OS Commands

Operating system-level commands and shortcuts for efficient system navigation.

## What's Included

### `Ubuntu.md`
Ubuntu/GNOME keyboard shortcuts and commands:
- **Window management** (snap, maximize, minimize)
- **App switching** (Alt+Tab, workspace switching)
- **Virtual desktops** (workspaces)
- **Search and launch** (Super key)
- **Multi-monitor** workflows
- **System debugging** tools

## Philosophy

> One task = one workspace. One workspace = one context.

Efficient OS usage means:
- **Keyboard-first**: Minimize mouse usage
- **Workspace organization**: Use virtual desktops for context switching
- **Intentional navigation**: Know what you're looking for
- **System awareness**: Understand what's running and why

## Quick Reference

### Window Management
- `Super + ← / →`: Snap window left/right
- `Super + ↑`: Maximize window
- `Super + ↓`: Restore/minimize
- `Super + Shift + ← / →`: Move window between monitors

### Workspaces
- `Super`: Open overview
- `Super + PageUp/PageDown`: Switch workspaces
- `Super + Shift + PageUp/PageDown`: Move window to workspace

### App Switching
- `Alt + Tab`: Switch between apps
- `Alt + \``: Switch windows within same app
- `Super + Tab`: App switcher (GNOME style)

## Tips

1. **Organize by context**: One workspace per project/task
2. **Use search**: `Super` key is faster than navigating menus
3. **Multi-monitor**: Use one monitor per context, not per app
4. **Learn system tools**: `gnome-system-monitor`, `htop`, etc.
5. **Customize shortcuts**: Adjust to your workflow in Settings
