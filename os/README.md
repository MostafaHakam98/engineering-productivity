# Operating System Tools

OS-level keyboard shortcuts, commands, and workflows for efficient system navigation.

## What's Included

### [ubuntu/](./ubuntu/) - Ubuntu/GNOME
Ubuntu and GNOME desktop environment shortcuts and commands.

## Structure

This directory is organized by operating system:
- `ubuntu/` - Ubuntu/GNOME shortcuts and commands

## Adding New Operating Systems

To add tips for other operating systems:
1. Create a new directory: `mkdir os/<os-name>`
2. Add a README.md with OS-specific documentation
3. Add markdown files with shortcuts and commands

## Philosophy

- **Keyboard-first**: Minimize mouse usage
- **Workspace organization**: Use virtual desktops for context switching
- **Intentional navigation**: Know what you're looking for
- **System awareness**: Understand what's running and why
