## Philosophy

> Tabs are cheap. Attention is not.

The goal is to minimize mouse usage and maximize keyboard efficiency:
- **Keyboard first**: Learn shortcuts, avoid mouse
- **Intentional navigation**: Know where you're going before you go
- **Clean workspace**: Close tabs you don't need
- **Context switching**: Use workspaces/tabs for different tasks

## Quick Reference

### Essential Shortcuts
- `Ctrl + T`: New tab
- `Ctrl + W`: Close tab
- `Ctrl + Tab`: Next tab
- `Ctrl + L`: Focus address bar
- `Ctrl + Shift + T`: Reopen closed tab
- `Ctrl + 1-9`: Jump to tab number

### Developer Tools
- `F12`: Open DevTools
- `Ctrl + Shift + I`: Open DevTools
- `Ctrl + Shift + J`: JavaScript console
- `Ctrl + Shift + C`: Inspect element

## Tips

1. **Learn incrementally**: Master 5-10 shortcuts first, then expand
2. **Use address bar for everything**: Search, navigate, run commands
3. **Tab management**: Keep related tabs together, close unused ones
4. **Bookmark bar**: Use keyboard shortcuts to access bookmarks
5. **Extensions**: Consider extensions that enhance keyboard navigation
